发表于2026-02-25
Embedding 是连接非结构化数据与深度学习模型的核心桥梁：它将离散的自然语言符号映射到连续的向量空间，使得语义或特征相似的对象在空间中具备可度量的邻近性。 在 Transformer 体系中，Word Embedding 是输入层的基石：它完成从 token 到向量的转换，为后续注意力机制与前馈网络提供可计算的语义表示。然而，传统词级嵌入面临 OOV（未登录词）、词表膨胀等问题，因此现代大模型普遍采用 Tokenizer + 子词分词（如 BPE） 来构建更高效的词表与嵌入层。 本文将系统梳理 Word Embedding 的技术演进，深入解析 Tokenizer 与 BPE 算法的原理，并提供完整的代码实现，帮助读者构建从文本预处理到模型输入的全链路认知。 从“符号鸿沟”到“语义空间” 此前的经典方法：从统计到预训练 现代 NLP 预处理：Tokenizer 与 BPE 算法 Transformer 中的 Word Embedding 原理 完整代码实现（从文本到 Embedding 输出） 现代演进与重点解读 拓展动手实现 总结与展望
